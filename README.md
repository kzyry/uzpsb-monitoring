# Прототип системы актов мониторинга кредитов

## Описание
Интерактивный прототип веб-интерфейса для создания и заполнения актов мониторинга кредитов на оборотные средства в системе HIVE Monitoring.

## Технологии
- **Vue.js 3** - JavaScript фреймворк
- **Vuetify 3** - Material Design компоненты
- **Material Design Icons** - иконки (локальные SVG файлы)
- **HTML5/CSS3** - разметка и стили

## Функциональность прототипа

### Реализованные функции:
✅ **13 разделов формы** организованных в табы:
   - Заголовок акта
   - Общая информация
   - Информация по кредитам (с динамической таблицей)
   - Связанные предприятия (с динамической таблицей)
   - Финансовое положение (автозаполняемые данные)
   - Долговая нагрузка (автоматические расчеты)
   - Обороты расчетного счета
   - Отчет по НДС (с динамической таблицей)
   - Адрес деятельности (в разработке)
   - Картотека-2 (в разработке)
   - Исполнение решений (в разработке)
   - Бизнес-план (в разработке)
   - Обеспечение (в разработке)

✅ **Интерактивные элементы**:
   - Переключение между табами
   - Добавление/удаление строк в таблицах
   - Валидация обязательных полей
   - Расчет автоматических полей
   - Индикатор прогресса заполнения
   - Автозаполнение тестовыми данными

✅ **UI/UX элементы**:
   - Минималистичный Material Design
   - Боковое меню навигации
   - Кнопки действий (Сохранить, Сбросить, Автозаполнение, Сформировать акт)
   - Уведомления (snackbar)
   - Визуальная индикация readonly полей
   - Цветовая индикация кредитной нагрузки

✅ **Адаптивность**:
   - Полная респонсивность для всех устройств
   - Оптимизация под планшеты (основное устройство)
   - Мобильная версия с горизонтальной прокруткой таблиц

## Как запустить

### Вариант 1: Простой запуск
1. Откройте файл `index.html` в браузере (двойной клик)

### Вариант 2: С локальным сервером (рекомендуется)

#### Используя Python:
```bash
cd prototype_monitoring
python3 -m http.server 8000
# Откройте http://localhost:8000 в браузере
```

#### Используя Node.js:
```bash
cd prototype_monitoring
npx serve .
# Откройте указанный URL в браузере
```

#### Используя VS Code:
- Установите расширение "Live Server"
- Правый клик на `index.html` → "Open with Live Server"

## Тестирование функционала

### Проверка автозаполнения:
1. Нажмите кнопку "Автозаполнение"
2. Проверьте, что поля заполнились тестовыми данными

### Проверка валидации:
1. Попробуйте оставить обязательные поля пустыми
2. Введите отрицательные значения в числовые поля
3. Введите неверный формат ИНН

### Проверка таблиц:
1. Перейдите на вкладку "Информация по кредитам"
2. Нажмите "Добавить строку"
3. Заполните данные
4. Удалите строку кнопкой удаления

### Проверка адаптивности:
1. Измените размер окна браузера
2. Откройте инструменты разработчика (F12)
3. Включите режим эмуляции мобильного устройства

## Доступные действия

| Кнопка | Действие |
|--------|----------|
| **Сохранить** | Показывает уведомление об успешном сохранении (эмуляция) |
| **Сбросить** | Сбрасывает изменения в форме |
| **Автозаполнение** | Заполняет форму тестовыми данными |
| **Сформировать акт** | Показывает уведомление о формировании (эмуляция) |
